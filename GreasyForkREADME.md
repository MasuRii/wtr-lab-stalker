# WTR Lab Stalker

Turn the WTR-LAB logo into a user-search toggle for profile lookup from the navbar.

## Installation

1. Install a userscript manager such as ScriptCat, Violentmonkey, Tampermonkey, or Stay.
2. Install the script from Greasy Fork: https://greasyfork.org/en/scripts/576097-wtr-lab-stalker
3. Visit WTR Lab and refresh any already-open matching pages after installation.

## Current Version

- Package: `wtr-lab-stalker`
- Version: `0.2.5`

## Usage

Open WTR Lab, toggle the logo, and confirm user search works while normal search returns when disabled.
